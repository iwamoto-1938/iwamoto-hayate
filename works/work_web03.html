作品名 ポートフォリオサイト作成
制作期間 89時間（約2ヶ月）
担当作業 デザイン、コーディング
制作の目的 自分の作品や人なりを見易くまとめ、採用担当者に見ていただくことで自分自身をアピールする
ターゲット 採用担当者の方々
作品説明 採用担当者に見やすいと思うかどうかを考え、トップページには簡潔な自己紹介と作品一覧にしました。
ファーストビュには印象に残りやすいよう、写真と自身の目標を明記し、白塗りの上に文字をおくようにすることで可読性も確保しました。
作品一覧はBootstrapのグリッドを使用し、きれいに整って見えるようにしました。
またサイト全体に動きをつけ、ひとつひとつに注目してもらえるよう、考えました。
使用ツール Photoshop,XD
開発ツール Visual Studio Code
使用言語 HTML,CSS,Sass,JavaScript
作品データ XDデータ
